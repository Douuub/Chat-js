
# Chat-JS

```javascript

│   ├── chat.css

│   ├── styles.css
├── server/

│   ├── connections.json

│   ├── models/
│   │   ├── User.js
│   │   └── Message.js
│   ├── routes/
│   │   └── message.js
├── package.json
└── package-lock.json
```





## How to use

Clone the project

```bash
  git clone https://github.com/Douuub/Chat-js
```

Go to the project directory

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```


## Tech Stack

**Client:** JavaScript, Html, CSS

**Server:** Node, JavaScript
